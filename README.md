# 📊 Data Visualization and Storytelling – Power BI

## Project Overview

This project was completed as part of **Task 2: Data Visualization and Storytelling** for a Data Analyst Internship.

The objective was to create an interactive Power BI dashboard that presents meaningful business insights using sales data. The dashboard focuses on clear visualizations and data storytelling to help users understand sales performance and make informed decisions.

## Dataset

* **Dataset:** Sales_Dataset (2).csv
* **Type:** Sales Dataset
* **Tool Used:** Microsoft Power BI

## Dashboard Visualizations

The dashboard includes:

* KPI Cards (Total Sales, Total Profit, Total Orders)
* Sales Trend Analysis
* Sales by Category
* Profit by Region
* Sales by Customer Segment
* Top Performing Products
* Interactive Filters and Slicers

## Key Insights

* Identified the highest-performing product categories.
* Compared sales and profit across different regions.
* Analyzed customer segments contributing the most sales.
* Observed sales trends over time.
* Created an interactive dashboard for easy business analysis.
